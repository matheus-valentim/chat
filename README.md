## 💻 Dependências

`express: 4.17.1`
<br>
`nodemon: 2.0.15`
<br>
`socket.io: 4.2.0`

## ☕ Usando o Chat

Para usar o Chat, siga estas etapas:

1. Após entrar no arquivo digite "npm start" no terminal do seu editor de texto.
2. Entre no navegador e pesquise por "http://localhost:3000/" e está pronto para uso.
